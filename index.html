  • git diff: muestra diferencias
  • para eliminar del git: git restore --staged index.html
  • esto restaura lo que hiciste, en este caso elimine este archivo y lo restaure con esto: git restore "index.html"
  • renombrar archivo: git mv "index.js" "script.js"
  • git log: los commits que has hecho
  • git log --oneline: muestra los commits pero en una sola linea
  • para renombrar el ultimo commit: git commit -m "nombre commit" --amend
  • muestra el log pero mas detallado (presionando q sales):git log -p
  • regresar el commit pero bienr raro: git reset (pones el codigo del commit en el que quieres regresar)

  • pasar los datos de una rama a la principal: git merge -m "merge back to the main" otraRama
  • git branch -d otraRama: elimina la otra rama
